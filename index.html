Google Drive Setup

Connect to Google Drive so PDF reports are automatically saved when you click "Save PDF to Drive". The History tab will show all saved PDFs.
Before testing: Make sure your Apps Script is deployed with Execute as: Me and Who has access: Anyone, and that you created a New Version after pasting the script.

How to get your Drive Folder ID

1. Open Google Drive and create a folder called Site Reports (or any name).
2. Open the folder. Copy the ID from the URL:
   drive.google.com/drive/folders/THIS_IS_YOUR_FOLDER_ID
3. Paste the Folder ID above.

Apps Script Code

Step 1 - Open script.google.com, create a New Project, paste the script below.
Step 2 - Replace YOUR_FOLDER_ID with your folder ID.
Step 3 - Deploy as Web App: Execute as Me, Access Anyone.
Step 4 - Copy the Web App URL, paste above, click Save then Test Connection.
Note: After any script edits, always create a New Version when redeploying.
